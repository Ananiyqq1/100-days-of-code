# 100 Days of Code

A personal challenge to code every day for 100 consecutive days, focusing on problem-solving and consistency.

## Project Structure

The project is organized by days:
- `Day-XX/`: Contains daily coding problems and their respective solutions.
    - `problem.md`: Description of the daily task.
    - `solution.c`: The implementation of the solution.

## Goals

- Maintain coding consistency.
- Improve low-level programming skills (C).
- Document progress and problem-solving techniques.
